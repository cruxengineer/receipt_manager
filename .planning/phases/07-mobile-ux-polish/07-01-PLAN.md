    1. In index.html, update the viewport meta tag to add `viewport-fit=cover`:
       ```html
       <meta name="viewport" content="width=device-width, initial-scale=1.0, viewport-fit=cover" />
       ```

    2. In src/index.css, inside the existing `@layer base` block (after the `:root` block), add global iOS fixes:

       ```css
       /* iOS Safari: remove blue/grey tap highlight flash on all tappable elements */
       * {
         -webkit-tap-highlight-color: transparent;
       }

       /* Prevent double-tap zoom on interactive elements and price numbers */
       button,
       [role="button"],
       input,
       .touch-none {
         touch-action: manipulation;
       }

       /* Prevent rubber-band overscroll on full-screen locked views */
       html,
       body {
         overscroll-behavior: none;
       }
       ```

    3. After the `@layer base` block (outside it), add safe area utility classes using plain CSS (not @layer — Tailwind v4 custom utilities go in @layer utilities):

       ```css
       @layer utilities {
         .pt-safe {
           padding-top: env(safe-area-inset-top);
         }
         .pb-safe {
           padding-bottom: env(safe-area-inset-bottom);
         }
         .pt-safe-4 {
           padding-top: max(env(safe-area-inset-top), 1rem);
         }
         .pb-safe-6 {
           padding-bottom: max(env(safe-area-inset-bottom), 1.5rem);
         }
       }
       ```

    These utilities will be applied to screen components in Task 2.
  </action>
